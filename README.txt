1. Clone project from repository
2. Install python 3.8 using pyenv (pyenv install <version>)
3. Install and create virtualenv with python 3.8 (virtualenv <name>) and add it as a interpreter in project
4. Install pluggins from requirements.txt
5. Run your code
6. Visit http://localhost:5000//api/v1/hello-word-2
7. To check python version write python --version in terminal
